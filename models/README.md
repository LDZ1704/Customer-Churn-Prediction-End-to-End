This folder stores trained artifacts generated by `train.py`.

- `best_model.joblib`: persisted sklearn/xgboost pipeline selected by ROC-AUC.
- `metadata.json`: schema, metrics, threshold, and column roles.
- `classification_report.txt`: text report on validation split.

Artifacts are created by running:
```
python train.py --data WA_Fn-UseC_-Telco-Customer-Churn.csv
```
